## Usage

1. Tap on one of the three signs: rock, paper, or scissors
2. See who won
3. Tap Play Again to play another game

## Compatibility

Hello World is a universal iOS app written with Swift 3.0, and built with Xcode 8.0 for iOS 10.0 or later.
